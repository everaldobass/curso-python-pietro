### Lógica

- A lógica é, de forma simples, a "gramática do pensamento". 
- É o estudo das regras que determinam se um raciocínio é válido ou não. 
- Ela serve para organizar as ideias de modo que as conclusões façam sentido a partir das informações que temos. [1, 2, 3] 
Existem três formas principais de olhar para ela:

   1. Lógica Formal: Foca na estrutura do argumento. Se eu digo que "Todo A é B" e "C é A", logo "C é B". Não importa o que são A, B ou C; se a estrutura estiver correta, o argumento é válido.
   2. Lógica Informal (ou Crítica): É a que usamos em debates e textos. Ela foca no conteúdo e no contexto, analisando se os argumentos são fortes, se há falácias (erros de raciocínio) ou se a conclusão é provável.
   3. Lógica de Programação: É a base da computação. É a sequência de passos lógicos (algoritmos) que dizem ao computador exatamente o que fazer para resolver um problema. [4, 5, 6, 7, 8] 

### Para que serve?
- Basicamente, para evitar que sejamos enganados por argumentos falsos e para nos ajudar a tomar decisões baseadas em evidências e coerência, em vez de apenas emoção ou intuição.
- Você quer entender como aplicar a lógica para identificar mentiras (falácias) ou está buscando a base para aprender programação?
